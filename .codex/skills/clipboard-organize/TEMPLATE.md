```yaml

title: "[文章标题]"

source: web-clipping
clipped_date: YYYY-MM-DD
original_url: [原始URL，如适用]

```

## 分类规则参考

### 内容类型判断

1. **教程/指南** (Tutorial):
   - 包含"如何"、"教程"、"指南"、"步骤"、"入门"等关键词
   - 有明确的步骤或操作说明
   - → 归类到 `30_研究/教程/` 或 `50_资源/教程/`

2. **研究/分析** (Research):
   - 包含"研究"、"分析"、"报告"、"调查"、"趋势"等关键词
   - 有数据、图表、深度分析
   - → 归类到 `30_研究/[主题]/`

3. **概念/知识点** (Knowledge):
   - 定义性内容，解释某个概念
   - 原子化的知识点
   - → 归类到 `40_知识库/[分类]/`

4. **新闻/动态** (News):
   - 时效性强，包含"最新"、"发布"、"更新"等关键词
   - 行业动态、产品发布
   - → 归类到 `50_资源/新闻/[领域]/`

5. **参考文章** (Reference):
   - 综合性文章，多个主题
   - 工具介绍、资源列表
   - → 归类到 `50_资源/文章/[主题]/`

### 标签生成规则

1. **主题标签**: 基于内容主题，如 #AI #前端 #生产力
2. **类型标签**: 基于内容类型，如 #教程 #研究 #新闻
3. **领域标签**: 基于所属领域，如 #技术 #生活 #工作
4. **难度标签**: 基于内容难度，如 #入门 #进阶 #专家
5. **来源标签**: 基于来源，如 #网络剪藏 #英文翻译
