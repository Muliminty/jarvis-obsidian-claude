## ⚙️ 配置原理

### 环境变量配置
Exa MCP 服务器通过 `EXA_API_KEY` 环境变量识别 API 密钥：
```bash
export EXA_API_KEY=your_exa_api_key_here
```

### MCP 服务器自动检测
当 Exa MCP 服务器检测到有效的 API 密钥时，会自动启用相应的高级工具。工具列表中的 `enabled` 字段会根据配置动态变化。
