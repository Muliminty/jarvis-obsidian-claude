## 使用技巧

### 最佳实践
1. **预处理剪藏**: 在剪藏时尽量保留完整标题和关键摘要
2. **定期整理**: 每天或每周使用本技能整理积累的剪藏
3. **质量检查**: 定期回顾归档内容，确保分类准确性
4. **反馈优化**: 遇到分类错误时，记录案例以便优化算法

### 高级功能
1. **批量处理**: 使用无输入模式处理多个文件
2. **自定义规则**: 根据需要修改`CLASSIFICATION_RULES.md`
3. **模板定制**: 调整`TEMPLATE.md`以适应个人偏好
4. **集成工作流**: 将本技能整合到日常知识管理流程中

### 故障排除
1. **分类不准确**: 检查内容摘要是否足够详细
2. **路径不存在**: 确认OrbitOS文件夹结构完整
3. **权限问题**: 检查文件读写权限
4. **技能未加载**: 确认技能文件夹在正确位置
